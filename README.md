## Recursos Principais

Esta aplicação Single Page Application (SPA), desenvolvida com Vue.js, foca em performance, responsividade e uma excelente experiência de usuário.

* **Tecnologias Modernas:** Construído com **Vue 3** (Composition API) e estilizado usando **Tailwind CSS** e **DaisyUI**.
* **Carregamento Lazy (Lazy Loading):** As rotas são carregadas sob demanda (lazy-loaded) usando o **Vue Router** para otimizar o tempo de carregamento inicial.
* **Animações e Usabilidade:** Inclui animações de scroll (`ScrollReveal`) e um botão fixo de **Scroll-to-Top** para melhor usabilidade.
* **Performance:** Implementação de um **Loading Screen** no `App.vue` para garantir que o conteúdo só seja exibido após o carregamento completo dos recursos.
